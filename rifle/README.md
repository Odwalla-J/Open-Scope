# Silent Scope Rifle

This release wouldn't feel complete without a compelling lightgun to match, and what better than a rifle inspired by Konami's own PSG-1 style arcade controller.
The model itself can be printed on all common 256x256x256 printes using roughly a kilogram of common PLA filament and supports all of the regular hardware you've come to expect from DIY lightguns (rumble, solenoid FFB, etc.) but also comfortably (and cartoonishly) houses the scope LCD with a supplemental magnifying lens for an ideal and premium feeling experience.
