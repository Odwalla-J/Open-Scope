# The Scope
The scope is a slightly insane solution that takes advantage of the ESP32-P4's High Speed USB OTG port to receive video streamed over serial from the host PC. 

This means 60fps 240x240 video rendered with a roughly 30ms delay either from MAME or PC cursor tracking for PCSX2 support.
